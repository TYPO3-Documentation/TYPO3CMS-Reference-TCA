PreviewRenderer examples
------------------------

Demonstrates property: :confval:`types-previewRenderer`.

This specifies the preview renderer only for records with the c :php:`blogexample_list`
as determined by the field `Ctype` of the table `tt_content`.

This could be used to preview the blog plugin in the :guilabel:`Content > Layout` module.

..  code-block:: php

    $GLOBALS['TCA']['tt_content']['types']['blogexample_list']['previewRenderer']
        = \MyVendor\MyExtension\Preview\PreviewRenderer::class;
