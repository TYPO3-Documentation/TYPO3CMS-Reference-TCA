..  _columns-color-properties-valuePicker:

..  confval:: valuePicker
    :name: color-valuePicker
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']
    :type: array
    :Scope: Display

    Renders a select box with static values next to the input field. When an
    item is selected, the value replaces the current input value.

    ..  versionchanged:: 14.0
        It is now recommended to define associative array keys for
        :php:`items` using :php:`label` and :php:`value`. Existing numeric
        arrays are automatically migrated during TCA processing.

        The Value Picker always behaves like a replace operation. Previous
        :php:`mode` variants like :php:`append` and :php:`prepend` were removed. See also
        `Breaking: #107677 - Drop prepend and append modes from TCA Value Picker <https://docs.typo3.org/permalink/changelog:breaking-107677-1760339783>`_.


    items (array)
        A list of selectable items. Each item should be an associative array:
        :php:`['label' => 'Shown label', 'value' => '#RRGGBB']`.
        The label can be plain text or a
        `label reference <https://docs.typo3.org/permalink/t3coreapi:label-reference>`_.
        The :php:`value` must be a hex color with exactly 7 characters,
        including the leading hash (for example, :php:`'#FF8700'`).
