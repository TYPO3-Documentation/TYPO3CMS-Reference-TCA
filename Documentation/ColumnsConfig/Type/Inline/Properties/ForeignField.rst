foreign\_field
==============

:aspect:`Datatype`
    string

:aspect:`Scope`
    Display / Proc.

:aspect:`Description`
    The :code:`foreign_field` is the field of the child record pointing to the parent record. This defines
    where to store the uid of the parent record.
