================
symmetric\_label
================

   :type: string   :Scope: Display / Proc.
    If set, it overrides the label set in :php:`$GLOBALS['TCA'][<foreign_table>]['ctrl']['label']` for the
    inline-view and only if looking to a symmetric relation from the "other" side.
