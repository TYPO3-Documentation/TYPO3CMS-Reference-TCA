====
size
====

:aspect:`Datatype`
    integer

:aspect:`Scope`
    Display

:aspect:`Description`
    Only useful in combination with :ref:`foreign\_selector <columns-inline-properties-foreign-selector>`.

    If set to 1 (default), the combination box is a select drop-down, else a select box of given size.
