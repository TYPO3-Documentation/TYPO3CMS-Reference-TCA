..  versionchanged:: 14.0

    The following TCA field  level search configuration options have been
    **removed**:

    * :php:`search.case`
    * :php:`search.pidonly`
    * :php:`search.andWhere`

    They have not been supported by the Core or common Extensions.

    See: `Breaking: #106976 - Removal of TCA search field configuration options <https://docs.typo3.org/permalink/changelog:breaking-106976-1750865819>`_
