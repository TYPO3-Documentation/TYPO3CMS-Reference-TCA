..  _columns-radio-properties-items:

..  confval:: items
    :name: radio-items
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']
    :required: true
    :type: array
    :Scope: Display  / Proc.

    An array of values which can be selected.

    Each entry is in itself an associative array.

    label (plain text label or `label reference <https://docs.typo3.org/permalink/t3coreapi:label-reference>`_)
        The displayed title.

    value (integer or string)
        The value stored in the database.
