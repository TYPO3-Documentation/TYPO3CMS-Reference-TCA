..  confval:: cols
    :name: json-cols
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']
    :type: integer
    :Scope: Display
    :default: 30

    Abstract value for the width of the :code:`<textarea>` field in which the
    JSON is displayed. To set the textarea to the full width
    of the form area, use the value 50.

    .. todo: add link after added to TSconfig reference

    This value can be overridden by page TSconfig.
