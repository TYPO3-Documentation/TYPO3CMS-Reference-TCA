..  confval:: size
    :name: country-size
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']['size']
    :type: integer
    :Scope: Display
    :Default: 1

    If set to 1 (default), displays a select drop-down, else a
    select box of given size.
