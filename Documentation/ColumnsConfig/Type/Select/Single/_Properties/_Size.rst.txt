..  confval:: size
    :name: select-single-size
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']
    :type: integer
    :Scope: Display
    :Default: 1

    If set to 1 (default), displays a select drop-down, else a
    select box of given size.
