..  confval:: itemsProcessors
    :name: select-checkbox-itemsProcessors
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']['itemsProcessors']
    :type: array
    :Scope: Display / Proc.

    ..  versionadded:: 14.0

    An array indexed numerically. Each entry defines an item processor implementing
    :php-short:`\TYPO3\CMS\Core\DataHandling\ItemsProcessorInterface`.

    Item processors are executed sequentially in ascending order of their numerical
    array key. Each processor can add, modify, or remove items from the
    :php-short:`\TYPO3\CMS\Core\Schema\Struct\SelectItemCollection`, influencing
    which items are ultimately displayed.

    This option can be combined with
    `itemsProcFunc <https://docs.typo3.org/permalink/t3tca:confval-select-checkbox-itemsprocfunc>`_.
    If both are configured, `itemsProcFunc` is executed first and `itemsProcessors`
    are executed afterwards.

    For fields of type `select`, each
    :php-short:`\TYPO3\CMS\Core\Schema\Struct\SelectItem` must be created with
    `type: 'select'` and must provide a `label` and a `value`. The value can be
    an integer or a string.

    Use the special value `'--div--'` to insert a divider (that cannot be selected).

    You can also use any arguments supported by the
    `items <https://docs.typo3.org/permalink/t3tca:confval-select-checkbox-items>`_
    array, including `group`, `icon` and `description`.
