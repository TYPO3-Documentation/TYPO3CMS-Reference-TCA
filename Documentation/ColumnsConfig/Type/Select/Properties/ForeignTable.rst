==============
foreign\_table
==============

.. confval:: foreign_table

   :type: string (table name)
   :Scope: Proc. / Display
   :RenderType: all

   The item-array will be filled with records from the table defined here. The table must be
   configured in :php:`$GLOBALS['TCA']`.
