=============================
disableNoMatchingValueElement
=============================

:aspect:`Datatype`
   boolean

:aspect:`Scope`
   Display

:aspect:`RenderType`
   all

:aspect:`Description`
   If set, then no element is inserted if the current value does not match any of the existing elements. A
   corresponding options is also found in Page TSconfig.
