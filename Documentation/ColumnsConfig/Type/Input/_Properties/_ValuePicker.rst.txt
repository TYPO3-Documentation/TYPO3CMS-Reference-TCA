..  _columns-input-properties-valuePicker:

..  confval:: valuePicker
    :name: input-valuePicker
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']
    :type: array
    :Scope: Display

    Renders a select box with static values next to the input field. When an
    item is selected, its value replaces the current input value.

    ..  versionchanged:: 14.0
        The :php:`mode` option no longer supports :php:`append` or
        :php:`prepend`. Value Picker always behaves like a replace operation.
        Any legacy :php:`mode` setting is ignored. See also
        `Breaking: #107677 - Drop prepend and append modes from TCA Value Picker <https://docs.typo3.org/permalink/changelog:breaking-107677-1760339783>`_.

        It is now recommended to define associative array keys for the
        :php:`items` configuration of the TCA configuration :php:`valuePicker`.
        The recommended keys are: :php:`label` and :php:`value`. Existing
        numeric array configurations will be automatically migrated to the new
        associative format during TCA processing.

    items (array)
        A list of selectable items. Each item should be an associative array:
        :php:`['label' => 'Shown label', 'value' => 'storedValue']`.
        The label can be plain text or a
        `label reference <https://docs.typo3.org/permalink/t3coreapi:label-reference>`_.
