    :name: flex-ds
    :Path: $GLOBALS['TCA'][$table]['columns'][$field]['config']
    :type: string
    :Scope: Display  / Proc.

    ..  versionchanged:: 14.0
        Multiple entries are no longer supported. Instead use
         `columnsOverrides  <https://docs.typo3.org/permalink/t3tca:confval-types-columnsoverrides>`_.

        See `Breaking: #107047 - Remove pointer field functionality of TCA flex <https://docs.typo3.org/permalink/changelog:breaking-107047-1751982363>`_
        for migration.

    This field can either contain a file reference to the file containing
    the FlexForm's XML (`'ds' => 'FILE:EXT:news/Configuration/FlexForm.xml'`)
    or contain the XML content directly (`'ds' => '<T3DataStructure>...'`).
