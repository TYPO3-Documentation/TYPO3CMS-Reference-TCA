:aspect:`Datatype`
    array

:aspect:`Scope`
    fieldWizard

:aspect:`Description`
    **Only with internal\_type='file'**

    Render a list of allowed file extensions. Enabled by default with `type='group'` and
    `internal_type='file'`.

.. warning::
   The configuration of `internal_type='file'` is deprecated since TYPO3 v9 and should not be used.
