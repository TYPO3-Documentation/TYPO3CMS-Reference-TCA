allowedExtensions
~~~~~~~~~~~~~~~~~

:aspect:`Datatype`
    string (list of file extensions)

:aspect:`Scope`
    Proc. / Display

:aspect:`Description`
    List of image types (by file extension) which can be cropped. Defaults to
    `$GLOBALS['TYPO3_CONF_VARS']['GFX']['imagefile_ext']` which is usually `gif,jpg,jpeg,png`.
